# Day 4: Frameworks (LangGraph, AutoGen, CrewAI)

## Learning objectives

- Spike **one** framework with a ≤300 line demo matching your orchestration diagram.
- Record **vendor lock-in** risks and escape hatch (plain Python fallback).

## Hands-on

1. Install chosen framework in a **venv** (separate optional.txt if heavy).
2. Implement hello-world multi-agent flow from their tutorial adapted to your domain nouns.
3. Write **decision record**: pick framework vs raw Python for production.

## Concepts

- **Debugging:** frameworks hide prompts — enable verbose logging / LangSmith-style tracing when possible.

## Done when

- [ ] Spike code in `labs/week13/spike_<framework>.py` or notebook.
