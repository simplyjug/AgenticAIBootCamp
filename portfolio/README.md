# Portfolio layer (your fork)

This folder is for **your** story after you fork the bootcamp.

## Suggested files

- `NOTES.md` — what you built, metrics, time spent, links to PRs  
- `demos/` — screenshots, short screen recordings (optional)  
- `results/` — eval tables, load-test summaries (no secrets)
